# To-Do List Web App

A responsive To-Do List web application built using HTML, CSS, and JavaScript with light and dark theme support.

## Features
- Add new tasks
- Delete tasks
- Light and Dark theme toggle
- Responsive UI
- Clean and minimal design

## Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation)
